NANDRAD::IDGroup Class Reference

Detailed Description

An IDGroup references one or more IDs and handles the encoding/decoding of an IDGroup-string.

// supported encodings
string severalIDs = "16,20,22";
string allIDs = "*";
string idInterval = "5-10,50-60";
string mixedIDs = "1,3,55,5-10";
// decode with IDGroup
IDGroup grp;
grp.setEncodedString(severalIDs);
// encode string
grp.encodedString();
